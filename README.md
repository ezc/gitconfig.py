    <b>Gitconfig class and gitconfig variable for ~/.gitconfig</b>

#### Install

`[sudo] pip install gitconfig`

```
from gitconfig import *

gitconfig.sections
>>> [...]

gitconfig.user.password
>>> 'secret'

gitconfig.not_existing
>>> None
```
